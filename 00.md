3. Add Kubectl Autocomplete:
    ```
    $ source <(kubectl completion zsh) 
    $ echo alias k=kubectl >> ~/.zshrc
    $ echo "if [ $commands[kubectl] ]; then source <(kubectl completion zsh); fi" >> ~/.zshrc 
    ```
